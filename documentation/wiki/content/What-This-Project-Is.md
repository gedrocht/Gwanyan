# What This Project Is

This project is a browser-based 3D scene.

It shows:

- a large square of dirt,
- a field of grass growing from that dirt,
- and wind that is created by your mouse movement.

When you move the mouse:

- the pointer position determines where the wind source is,
- the speed of the mouse movement determines how strong that wind is,
- and the grass bends, sways, and settles back upright because a spring simulation is running.

This means the app is not just drawing a picture. It is constantly calculating motion.
