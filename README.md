# Gwanyan

This repository is bootstrapped with a strict GitHub-first quality and security baseline.

## Included protections

- CI checks for workflow validity and repository hygiene
- Secret scanning with Gitleaks
- Dependency review on pull requests
- CodeQL static analysis
- OpenSSF Scorecards analysis
- Dependabot for action and package update visibility
- CODEOWNERS, contribution guidance, and security reporting instructions

## Next recommended steps

1. Add the project source code and language-specific test suite.
2. Enable branch protection on `main` and require the status checks from `.github/workflows/`.
3. Turn on GitHub security features available to your plan, including secret scanning and Dependabot alerts.
